# DeepSurv-Cancer-Survival-Analysis
DeepSurv-based survival analysis for cancer progression (METABRIC dataset)

This repository contains the implementation of a DeepSurv-based survival analysis framework for cancer progression prediction using the METABRIC dataset.

## Features
- DeepSurv neural survival model
- Comparison with CoxPH, RSF, and Survival-SVM
- Autoencoder-based feature learning
- Integrated Brier Score and time-dependent AUC evaluation
